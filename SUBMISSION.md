# 点球宇宙 Penalty Oracle 参赛草稿

## 项目名称

点球宇宙 Penalty Oracle

## 项目简介

一个世界杯点球心理剧本生成器。用户选择球员人格、助跑方式、射门方向、压力指数和门将读心，点击“开罚”后生成一份赛后小报，包括进球率、梗指数、VAR 戏份、时间线和可下载 PNG 海报。

## 项目类型

网页 / H5 / 互动小游戏

## 项目说明

核心玩法：

每天不用登录，也不用联网后端。打开页面后，用户可以直接调一组点球参数：谁来罚、怎么助跑、踢向哪里、门将怎么猜、现场压力有多大。点击“开罚”后，系统会用一套简单的本地规则生成结果，不追求真实预测，而是把世界杯点球瞬间变成一个有戏剧感、有梗感、能分享的小报。

我做这个项目时想解决的是“足球梗的参与门槛”。很多人不一定知道复杂战术，但都能理解点球前的紧张、门将猜方向的心理战、VAR 加戏和赛后评论区的热闹。所以它不需要专业足球知识，10 秒内就能玩懂。

技术上它是一个完全静态的网页：没有服务器、没有数据库、没有云存储。最近三脚点球只保存在浏览器本地 `localStorage`，小报图片也在本地 canvas 生成并下载。部署时只要把 `index.html`、`styles.css`、`app.js` 放到 GitHub Pages 就能运行。
