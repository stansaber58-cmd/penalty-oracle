# 点球宇宙 Penalty Oracle

一个世界杯主题互动产品：用户选择球员人格、助跑方式、射门方向、压力指数和门将读心，系统会生成一份带进球率、梗指数、VAR 戏份和赛后小报的点球剧本。

## 玩法亮点

- 点球心理剧本：把一脚点球拆成性格、助跑、方向、压力和门将选择。
- 动态球场演算：点击开罚后，足球会沿着选择方向飞向球门。
- 赛后小报生成：自动生成标题、战报、时间线和可下载 PNG 海报。
- 本地档案：只用浏览器 `localStorage` 保留最近三脚点球，方便反复比较不同剧本。
- 零后端依赖：不需要服务器、不需要数据库、不需要云存储，也不需要登录。

## 运行方式

这是一个纯静态项目，直接打开 `index.html` 即可运行，也可以部署到 GitHub Pages。

## 参赛说明

这个项目想做的是一个“世界杯情绪模拟器”：不预测真实比分，而是把球迷最熟悉的点球瞬间变成可操作、可分享、可二创的小产品。用户可以很快生成一份属于自己的点球赛后小报，既有足球味，也有互联网梗感。
